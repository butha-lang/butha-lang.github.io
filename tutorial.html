Introduction

Welcome to the Butha language tutorial. This guide will help you get started with Butha, covering the basics of setting up your environment, writing your first program, and understanding the key features of the language.

Setup

To begin using Butha, you need to install the Butha interpreter and set up your development environment.

Installation

Download the Butha interpreter from the official website. Choose the installer suitable for your operating system and follow the installation instructions.

Setting Up the Environment

Ensure that the Butha executable is added to your system's PATH. This will allow you to run Butha commands from the terminal or command prompt.

Basic Syntax

Butha uses a clean and straightforward syntax designed for ease of use. Here's an overview of the basic syntax:

Variables

Variables in Butha are declared using the `let` keyword:


let x = 10;
let name = "Butha";

Data Types

Butha supports several data types, including integers, floats, strings, and booleans:


let age = 25;         // Integer
let height = 5.9;     // Float
let greeting = "Hello"; // String
let isActive = true; // Boolean

Functions

Functions are defined using the `function` keyword. Here's how you can define and call a function in Butha:


function greet(name) {
    return "Hello, " + name;
}
let message = greet("World");
print(message);

Control Structures

Butha includes common control structures such as conditionals and loops:

Conditionals


let x = 10;
if (x > 5) {
    print("x is greater than 5");
} else {
    print("x is 5 or less");
}

Loops


for (let i = 0; i < 5; i++) {
    print(i);
}

Modules

Butha allows you to use and create modules for code organization and reuse. Here's how to import and use a module:


import { Math } from "core_modules/math.butha";
let sum = Math.add(5, 3);
print(sum);

Modules can also be created to encapsulate and reuse code:


function add(a, b) {
    return a + b;
}
export { add };

Advanced Topics

Explore more advanced features of Butha, including concurrency, error handling, and custom modules:

Concurrency

Butha supports concurrency with its built-in concurrency module:


import { Concurrency } from "core_modules/concurrency.butha";
Concurrency.run(() => {
    print("Running in parallel");
});

Error Handling

Handle errors using try-catch blocks:


try {
    let result = 10 / 0;
} catch (error) {
    print("Error: " + error.message);
}

Troubleshooting

If you encounter issues while using Butha, refer to these common solutions:

Common Issues

  • **Syntax Errors:** Double-check your code for syntax mistakes.
  • **Module Import Errors:** Ensure the module paths are correct and the modules are properly installed.
  • **Runtime Errors:** Check the error messages for clues on what might be going wrong.
