Introduction

Butha is a high-level programming language designed for ease of use and flexibility. It features a clean syntax, rich set of core modules, and extensive libraries for various functionalities.

Getting Started

To get started with Butha, you need to install the Butha interpreter and set up your development environment.

Installation

Download and install the Butha interpreter from the official website. Follow the installation guide for your operating system.

Basic Usage

Write your Butha code in a file with the `.butha` extension. You can execute your code using the Butha interpreter with the command:

butha your_file.butha

Syntax

Butha syntax is designed to be intuitive and easy to learn. Here are some basic syntax rules:

Variables


let x = 10;
let name = "Butha";

Functions


function greet(name) {
    return "Hello, " + name;
}

Control Structures


if (x > 10) {
    print("x is greater than 10");
} else {
    print("x is 10 or less");
}

Core Modules

Butha provides several core modules to support fundamental programming tasks.

Core Module: Math

Provides basic mathematical operations and constants.


import { Math } from "core_modules/math.butha";
let result = Math.add(5, 3);

Core Module: IO

Handles input and output operations.


import { IO } from "core_modules/io.butha";
IO.print("Hello, World!");

Standard Library

The standard library provides additional functionality for common programming tasks.

Math Library

Includes advanced mathematical functions and constants.


import { Math } from "standard_lib/math.butha";
let pi = Math.PI;

Utils Library

Provides utility functions for type checking, formatting, and array manipulation.


import { Utils } from "standard_lib/utils.butha";
let formattedDate = Utils.formatDate(new Date());

Third-Party Libraries

Butha supports integration with various third-party libraries to extend its capabilities.

Axios

A promise-based HTTP client for making requests.


import { Axios } from "third_party/axios.butha";
let response = Axios.get("https://api.example.com/data");

Lodash

A utility library providing many helpful functions.


import { Lodash } from "third_party/lodash.butha";
let array = [1, 2, 3];
let doubled = Lodash.map(array, x => x * 2);

Examples

Here are some example scripts written in Butha:

Basic Example


let name = "World";
print("Hello, " + name);

Advanced Example


function factorial(n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}
let result = factorial(5);
print("Factorial of 5 is " + result);
